### テキストタグ

**1. レイアウト**

```html
ヘッダーの<div>要素
<div class="header">
</div>
```

```html
メインの<div>要素
<div class="main">
</div>
```

```html
フッターの<div>要素
<div class="futter">
</div>
```

メモ:
- レイアウトは<div>要素によって構成されている
- <div>要素のdivは、「division」の略
- 要素をグループ化するために使用される
